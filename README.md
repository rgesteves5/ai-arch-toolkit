## Quick Start

```bash
# Install uv (if you don't have it)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Clone and install
git clone <repo-url> && cd ai-arch-toolkit
uv sync --dev

# Run tests
uv run pytest

# Run an example
uv run python examples/01_hello_world.py
```
